# What is Libreboot / Coreboot

Coreboot is an extended firmware platform that delivers a lightning fast and secure boot experience on modern computers and embedded systems.
As an Open Source project it provides auditability and maximum control over technology.
https://www.coreboot.com

Libreboot is a modification of Coreboot that is only possible to be installed on a few select machines with specific hardware that allows it. It modifies Coreboot by removing all non-free software and disables the Intel AMT.
https://www.libreboot.org/
